About This Portal

A web application by B.Tech CSE students of K.R. Mangalam University designed to solve a real everyday campus problem — lost & found item management — using modern web technology.

Project Overview

Why We Built This

Replacing unreliable notice boards and word-of-mouth with a centralised, searchable digital platform.

🎯

Problem Statement

Students and staff frequently misplace belongings across classrooms, labs, the library, and hostels, with no structured system to report or track lost items on campus.

💡

Our Solution

A web platform where users can post lost/found items, search by keyword, category, or location, and connect directly — all in one centralised, free, accessible system.

🛠️

Tech Stack

Built with HTML5, CSS3 & vanilla JavaScript for the frontend, PHP for backend logic, and MySQL for data persistence — deployed locally via XAMPP.
HTML5 CSS3 JavaScript PHP MySQL XAMPP
✅

Key Features

User authentication, item posting with categories & locations, keyword & filter search, claim verification, email-based contact, and mobile-friendly responsive design.
